# CarCollection — Автомобили с аукционов под ключ

Сервис помощи в покупке автомобилей с аукционов США, Японии, Кореи и доставке в РФ.

---

## О проекте

CarCollection помогает частным лицам и предпринимателям покупать автомобили на аукционах Copart, IAAI, USS и других. Мы сопровождаем клиента на всех этапах: от подбора лота до выдачи ключей в РФ.

 *Прозрачный расчёт под ключ без скрытых платежей*

---

## Возможности

- Каталог актуальных лотов с фото и характеристиками
- Калькулятор полной стоимости «под ключ» (ставка + доставка + пошлины + комиссия)
- Фильтрация по странам (США, Япония, Корея)
- Детальная карточка автомобиля с полной разбивкой расходов
- Адаптивный дизайн для всех устройств
