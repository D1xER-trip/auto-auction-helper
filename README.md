# CARCOLLECTION

Профессиональный сервис по подбору и доставке автомобилей с аукционов США, Японии, Кореи  Проект создан для прозрачной покупки авто «под ключ» с полным расчётом всех расходов.

---

## Функционал

**Интерактивный каталог:** Актуальные лоты с аукционов Copart, IAAI, USS — от бюджетных седанов до премиальных внедорожников.

**Калькулятор «под ключ»:** Мгновенный расчёт полной стоимости с учётом ставки, доставки, таможни и комиссии.

**Умные фильтры:** Сортировка по стране, бюджету, типу повреждений и году выпуска.

**Инженерный дизайн:** Минималистичный интерфейс, вдохновленный технической документацией и аналитическими отчётами.

---

## Технологии

- **React 18 + Vite** — сверхбыстрая сборка и работа
- **React Router Dom** — бесшовная навигация
- **CSS Grid / Flexbox** — адаптивная и точная вёрстка
