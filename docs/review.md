# Optimization Plan — CarCollection

## Цели оптимизации
- Уменьшить время загрузки страниц до < 2 секунд
- Снизить нагрузку на сервер
- Улучшить Core Web Vitals (LCP, FID, CLS)

## Текущие метрики

| Метрика | Значение | Цель |
|---------|----------|------|
| Время загрузки главной | 1.8 с | < 1.5 с |
| Время загрузки каталога | 2.4 с | < 1.8 с |
| Размер бандла (JS) | 450 KB | < 300 KB |
| LCP (самая большая картинка) | 2.1 с | < 1.5 с |

---

## 1. Оптимизация изображений

**Проблема:** Картинки авто весят 1-2 МБ, загружаются долго.

**Решение:**
- Конвертировать в WebP (сжать на 40-50%)
- Добавить lazy loading: `<img loading="lazy" ... />`
- Использовать CDN (Cloudinary / Imgix)

**Ожидаемый результат:** Снижение времени загрузки на 30%
