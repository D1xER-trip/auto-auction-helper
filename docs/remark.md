# Замечания и доработки: CarCollection

## 1. Визуальные замечания

**Карточки авто в каталоге**
- На планшете картинки становились слишком узкими, текст не читался
- Сделали 2 колонки вместо 3, добавили больше воздуха между строками

**Фотографии машин**
- У разных лотов фото были разной высоты, из-за этого кнопки ехали вразнобой
- Зафиксировали высоту картинок, привели всё к единому виду

---

## 2. Что сломалось и починили

**Калькулятор**
- При смене страны курс доллара не менялся, цена считалась неверно
- Теперь курс подтягивается автоматически при выборе США/Японии/Кореи

**Сортировка и фильтры**
- Если сперва отфильтровать по стране, а потом отсортировать по цене — фильтр слетал
- Развязали эти две штуки, теперь работают независимо

**Форма отправки заявки**
- На айфоне выпадающие списки были серыми и уродливыми
- Перекрасили под общий стиль сайта

**Возврат из карточки товара**
- Когда возвращаешься из описания лота обратно в каталог, страница улетала наверх
- Добавили запоминание места, где остановился

## 3. Ускорение работы

**Шрифты**
- Стандартные шрифты грузились долго на слабых телефонах

**Калькулятор**
- При быстром наборе цифр цена пересчитывалась слишком часто
- Поставили задержку, теперь не дёргается при каждом символе
