# Growth Strategy — CarCollection

## Видение
CarCollection должен стать **крупнейшим сервисом по подбору и доставке авто с аукционов** в России.

## Этапы развития

### Phase 1: MVP (текущий) — Июнь 2026-2028
- ✅ Каталог лотов (статический)
- ✅ Калькулятор стоимости
- ✅ Контакты в Telegram
- 🟡 Админ-панель (в разработке)

### Phase 2: Рост — Июль-Август 2026
- [ ] Интеграция с API аукционов (Copart, IAAI)
- [ ] Авторизация через Google / Telegram
- [ ] Сохранение расчётов в личный кабинет
- [ ] Telegram-бот для уведомлений
- [ ] SEO-оптимизация (статьи, ключевые слова)

### Phase 3: Масштабирование — Сентябрь-Декабрь 2026
- [ ] Мобильное приложение (React Native)
- [ ] AI-подбор авто (рекомендации)
- [ ] Система отзывов с фото/видео
- [ ] Партнёрская программа

### Phase 4: Экосистема — 2027
- [ ] Свой аукцион
- [ ] Кредитование и страхование
- [ ] Выход на рынки СНГ
