1. Главная страница (Landing)
Старт → Экран приветствия
Пользователь может:

Ознакомиться с преимуществами сервиса (прозрачный расчёт, полный цикл).

Быстро рассчитать стоимость авто (виджет-калькулятор).

Перейти в каталог актуальных лотов.

Изучить информацию о услугах (доставка, таможня, подбор).

Сценарии:
Клик по кнопке «КАТАЛОГ ЛОТОВ» → Переход к списку автомобилей.

Клик по кнопке «РАССЧИТАТЬ» в виджете → Открытие расширенного калькулятора.

Клик по логотипу → Обновление главной страницы.

Клик по статье в блоге → Переход к гайду ("Как растаможить авто в 2026")


2. Каталог лотов (Аукционные автомобили)
Пользователь может:
Просматривать сетку доступных лотов (фото, название, аукцион).

Видеть текущую ставку и итоговую цену «под ключ».

Фильтровать по стране (США / Япония / Европа).

Сортировать по цене (дешевле / дороже).

Сценарии:
Выбор карточки «BMW X5 2021» → Переход на детальную страницу лота.

Применение фильтра «Япония» → Отображаются только японские аукционы.

Клик по кнопке «СНАЧАЛА ДЕШЕВЛЕ» → Ре-сортировка лотов по ставке.


3. Детальная страница лота (Карточка автомобиля)
Пользователь может:
Изучать фото, характеристики, повреждения, VIN.

Видеть разбивку стоимости «под ключ» (ставка → доставка → пошлины → комиссия).

Изменять сумму ставки в калькуляторе → мгновенный пересчёт итога.

Сохранять лот в «Избранное» (требуется авторизация).

Оставлять заявку на участие в торгах.

Сценарии:
Изменение поля «Ставка на аукционе» с 
12500
на
12500на13 000 → Итоговая цена пересчитывается автоматически.

Клик по кнопке «ОСТАВИТЬ ЗАЯВКУ» → Открытие чата с менеджером в Telegram.

Клик по «★ СОХРАНИТЬ В ИЗБРАННОЕ» → Если не авторизован → редирект на страницу входа.


4. Личный кабинет (Авторизованная зона)
Пользователь может:
Просматривать и управлять избранными лотами.

Отслеживать статусы своих заявок (в обработке / на торгах / выигран / в доставке).

Смотреть историю расчётов (сохранённые калькуляции).

Редактировать профиль (контакты, предпочтения).

Сценарии:
Переход в раздел «Мои заявки» → Видит заявку на BMW X5 со статусом «На рассмотрении».

Отмена избранного лота → Удаление из списка.

Клик по заявке → Открытие диалога с менеджером в Telegram или Max для уточнения деталей.





5. Калькулятор стоимости (Отдельный инструмент)
Пользователь может:
Выбрать страну аукциона (США, Япония, Европа, Корея).

Указать марку, модель, год, пробег, тип повреждений.

Получить финальную цену «под ключ» (включая все скрытые платежи).

Сохранить расчёт (на email или в личный кабинет).

Сценарии:
Выбор «США» → Подставляются средние ставки доставки ($1500–2500).

Выбор «Электромобиль» → Автоматически применяется льготная таможенная ставка (-30%).

Клик по «ОТПРАВИТЬ РАСЧЁТ» → Система отправляет PDF-файл на указанный email.



6. Административная панель (Менеджер)
Менеджер может:
Просматривать список заявок от клиентов (с фильтром по дате, статусу).

Назначать ответственного за заявку.

Изменять статус заявки (→ «На торгах» → «Выигран» → «В пути»).

Добавлять вручную лоты из аукционов, которые не подгрузились по API.

Редактировать тарифы калькулятора (пошлины, комиссии, курсы валют).

Сценарии:
Менеджер открывает панель → Видит красный бейдж «3 новые заявки».

Кликает на заявку #102 → Меняет статус с «Новая» на «Связались с клиентом».

Менеджер добавляет лот «Toyota Supra 2024» вручную → Лот появляется в каталоге через 5 секунд.



7. Блог / База знаний (Информационный раздел)
Пользователь может:
Читать статьи о покупке авто с аукционов.

Смотреть видеоинструкции (YouTube-вставки).

Подписаться на новости (email-рассылка).

Искать по тегам (#таможня, #доставка, #аукционы).

Сценарии:
Клик по статье «Как проверить авто по VIN бесплатно» → Открытие страницы с гайдом.

Ввод в поиске «растаможка электромобиля» → Система показывает релевантные статьи.

Клик по кнопке «ПОДПИСАТЬСЯ» → Запрос email → Добавление в Mailchimp / Telegram-канал.




8. Обратная связь / Поддержка (Telegram-интеграция)
Пользователь может:
Написать в техподдержку через Telegram (кнопка в футере).

Получить быстрый ответ от менеджера (в течение 15 минут в рабочее время).

Прикрепить скриншоты лота или документа.

Сценарии:
Клик по ссылке будет позжу → Открывается Telegram-чат с менеджером.

Пользователь пишет: «Хочу заказать BMW X5, лот #001» → Менеджер запрашивает контактные данные → Начинает оформление заявки.
